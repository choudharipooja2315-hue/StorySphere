# StorySphere

StorySphere is a web-based storytelling platform that allows writers to create and publish stories while enabling readers to explore and read content online. The platform provides user authentication, story management, chapter creation, and an interactive reading experience.

## Features

* User Registration and Login
* Writer Dashboard
* Create and Manage Stories
* Add Chapters to Stories
* View Story Details
* Reader-Friendly Interface
* Database Integration for Data Storage

## Tech Stack

### Frontend

* HTML
* CSS
* JavaScript

### Backend

* Python
* Flask

### Database

* MySQL

## Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/StorySphere.git
```

2. Navigate to the project directory

```bash
cd StorySphere
```

3. Install required dependencies

```bash
pip install -r requirements.txt
```

4. Configure database settings in the configuration file.

5. Run the Flask application

```bash
python app.py
```
